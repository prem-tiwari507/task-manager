# Task manager v1
Task manger application that supports CRUD operations.

Technoloy involed (with versions)- 
Python 3.11.2
Node.js - v20.19.3
Angular 20.0.3
SQLite - DB

Developer - 
1. Clone the repo.
2. Active python viturla env - venv\Scripts\activate
3. Install backend dependencies using reqiirement.txt
4. Install angular dependencies - npm install
